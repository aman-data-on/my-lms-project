# Lesson illustrations

Drop topic illustration images here. They are served at `/illustrations/<file>`.

## Required

- **`who-we-are.png`** — the isometric cloud + city illustration for
  Module 1 → 1.1 "Who We Are — Leapswitch Networks".
  Referenced by the lesson content (`illustration: "/illustrations/who-we-are.png"`).
  Until this file exists, the reader gracefully falls back to the built-in
  vector illustration, so nothing breaks.

Recommended: a transparent or white-background PNG/SVG, roughly 4:3, ≥ 800px wide.
